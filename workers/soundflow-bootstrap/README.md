# SoundFlow Bootstrap

This package is the minimal AI Audio Studio bridge for Pro Tools-oriented revision execution.

## Install

1. Copy this folder into your SoundFlow packages workspace on the Pro Tools operator machine.
2. Open SoundFlow and rescan or reload user packages.
3. Point AI Audio Studio revision execution at the generated JSON payload from `workers/revision-parser/main.py`.

## Actions

- `setFader`: reads `track` and `value_db`
- `comment`: logs the generated revision note into the SoundFlow console

## Usage

Feed the JSON file generated by the revision parser into `index.js` and call `run(payload)`.
This skeleton is intended as the operator-safe bootstrap layer for fuller Pro Tools automation.
