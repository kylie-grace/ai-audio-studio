This project is licensed under the GNU Affero General Public License v3.0 (AGPL-3.0).

You are free to:
- use this system in personal or commercial studio workflows
- make money using this system as part of your creative work
- modify and improve the system
- share your improvements under the same license

## Our Intent

This project is designed to support independent creators, engineers, and studios by reducing overhead and enabling more focus on creative work.

We strongly support:
- independent use
- learning and experimentation
- building sustainable creative businesses

## Please Do Not

While the license allows broad use, we ask that you **do not**:

- repackage and sell this system as a standalone product
- present this system as your own work without attribution
- build closed, proprietary versions without contributing back
- exploit this project in ways that harm the creative community

- a company building on top of this system
